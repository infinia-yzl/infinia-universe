# infinia-universe

## Quickstart:

#### npm
```
npm install
npm start
```

#### pnpm
```
pnpm install
pnpm start
```

## Deployment:
#### Option 1: Using the deploy script

> 1. Ensure that SSH is configured with GH account.
> 1. Run `deploy.sh`.


#### Option 2: Manually

> 1. `pnpm build`
> 1. Checkout `main` branch.
> 1. Commit the changes under `dist`.
> 1. Push to remote `main` branch.
